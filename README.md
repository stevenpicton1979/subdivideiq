# SubdivideIQ

Subdivision feasibility pre-screen for Brisbane homeowners.

Delivers a traffic light feasibility report in under 60 seconds, surfacing the constraints that consultants either don't tell you upfront or charge thousands to discover.

**Price:** $79 AUD

## Stack
- Vanilla HTML/CSS/JS
- Vercel serverless Node.js
- Supabase PostGIS (project: fzykfxesznyiigoyeyed)
- Stripe
- Resend
- Mapbox

## Local setup
```bash
cp .env.example .env
# Fill in credentials
npm install
vercel dev
```
